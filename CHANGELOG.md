# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-05-14

### Added
- GitHub Actions CI/CD pipeline for automated testing and deployment
- Enhanced TypeScript strict mode with strict null checks
- Improved error handling and logging throughout the application
- Input validation using Zod for all API endpoints
- Security audit integration in CI/CD pipeline

### Changed
- Upgraded to production-grade code quality standards
- Enhanced ESLint configuration for better code consistency
- Improved error messages and user feedback
- Updated dependencies to latest secure versions

### Fixed
- TypeScript compilation warnings resolved
- ESLint rule violations fixed
- Environment variable validation on startup

## [0.1.0] - 2026-05-08

### Added
- Initial release of Travel Flight Bot
- LINE Bot integration for flight search
- Supabase database for conversation state
- Next.js frontend for dashboard
- SerpApi integration for flight data
- Vercel Cron for scheduled searches
